## 项目分包结构
- components # 组件目录
- core # npm 包入口
- docs # 文档目录
- hooks # 组合式API hooks 目录
- play # 组件开发实验室
- theme # 主题目录
- utils # 工具函数目录

## components结构
- Button.vue 组件
- Button.test.tsx 组件测试用例
- type.ts 类型
- style.css 样式
- constants.ts 常量
- index.ts 入口文件

## node.js 版本限制
1. 确保已安装nvm并能正确运行
2. git bash进入项目根目录
3. 使用`nvm use ${cat .nvmrc}`命令切换版本
